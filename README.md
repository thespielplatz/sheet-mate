Features: See [Features](docs/FEATURES.md)

## How to use

### Docker

- Install docker

```bash
docker run thespielplatz/sheet-mate:latest
```

- Open http://localhost:3000

### From source

- Checkout this repo

```bash
npm i
npm run dev
```
